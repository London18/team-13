# Team Sweet Potato

## Setup for sending alert emails

- Gmail's mail servers are used to send email alerts. To be able to use this feature, a Gmail email account and its password must be provided. Furthermore, the Gmail account won't allow sending the email unless you go to your Google account settings and "Allow less secure apps".
